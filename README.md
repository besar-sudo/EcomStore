# EcomStore
A responsive e-commerce web application developed with HTML, CSS, PHP, and MySQL, focusing on server-side functionality and clean, practical design.

Features:
- Browse products by category
- Add and remove items from the shopping cart
- Secure checkout and order processing
- Admin panel to manage products, orders, and users
- Responsive design for desktop and mobile

Technologies Used:
- HTML and CSS for front-end design
- PHP for backend logic and form handling
- MySQL for database management

Installation:
1. Clone the repository: git clone https://github.com/besar-sudo/EcomStore.git
3. Set up a local server (XAMPP, WAMP, or LAMP)
4. Import the database file into MySQL
5. Update database connection settings in config.php
6. Open the project in your browser

Usage:
- Browse products and add items to the cart
- Admin can manage products, orders, and users
- Ideal for learning server-side development and building a real e-commerce website without JavaScript
